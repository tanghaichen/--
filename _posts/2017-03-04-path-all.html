安装需要

安装nodejs和nvm

  • 下载安装nodejs和nvm //在setting中加入 node_mirror: https://npm.taobao.org/mirrors/node/ npm_mirror: https://npm.taobao.org/mirrors/npm/
  • 安装cnpm || 通过npm install -g cnpm --registry=https://registry.npm.taobao.org //国内镜像
  • npm install -g nodemon //包管理器。可以及时更新

安装express和打包工具grunt

  • npm install express-generator -g

    安装express

    创建express模板 express myapp --view ejs express然后进入express生成的文件夹中, npm install 安装所需要的文件，最后 npm start

  • npm install grunt-cli -g

    软件打包工具

安装less和sass

  • npm install -g less

    安装less//lessc -h

  • npm install -g csslint

    npm install -g jshint

    安装sass

安装mongodb

  • mongodb下载安装

    ./bin/mongod.exe --dbpath /path/to/database --logpath /path/to/log-file --port 27017

安装git

  • 官网下载git安装包安装即可

安装Ionic

  • cnpm install -g cordova ionic
